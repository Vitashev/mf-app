# WebCompApp

The app demonstrates of microfrontend implementation using Angular with NgRx shared state between micro apps.

## Installling dependencies

`npm i`

## Run Shell app

`nx run shell:serve-mfe`

## Run Gallery app

`nx run gallery:serve`
